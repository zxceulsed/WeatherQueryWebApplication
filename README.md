# WeatherQueryWebApplication

## Quick Start

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/zxceulsed/WeatherQueryWebApplication
cd WeatherQueryWebApplication
```

### 2. Настройка переменных окружения

Скопируйте пример и заполните свои данные:

```bash
cp sample.env .env sample
```

Откройте `.env` и отредактируйте нужные переменные.

### 3. Сборка и запуск Docker контейнеров

```bash
docker compose up --build
```

### 4. Проект будет доступен по ссылке:

```
http://127.0.0.1:8000/
```
